# 节拍器 - 最简单使用方案

## ⭐ 推荐方案1：Kivy Launcher（最简单！无需打包！）

**不需要任何网络配置，10分钟就能在手机上运行！**

### 步骤：

1. **手机安装 Kivy Launcher**
   - 方式A：Google Play 搜索 "Kivy Launcher" 安装
   - 方式B：下载 APK 安装：https://github.com/kivy/kivy-launcher/releases

2. **准备项目文件**
   - 从电脑复制以下文件到手机：
     - `main.py`（节拍器主程序）
     - `sounds/` 文件夹（声音文件）
   
   - 在手机上创建目录：
     `/sdcard/kivy/节拍器/`
   
   - 把文件放进去：
     `/sdcard/kivy/节拍器/main.py`
     `/sdcard/kivy/节拍器/sounds/`

3. **手机上运行**
   - 打开 Kivy Launcher APP
   - 看到 "节拍器" 项目
   - 点击运行！

## 🌩️ 推荐方案2：GitHub Actions 在线打包（需要 GitHub 账号）

**不用配置本地环境，GitHub 免费帮你打包！**

### 步骤：

1. **注册 GitHub 账号**
   - https://github.com/

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 命名：metronome-app
   - 设为 Public 或 Private

3. **上传项目文件**
   - 把 `JPTool` 里的所有文件上传到仓库
   - 确保包含：
     - `main.py`
     - `buildozer.spec`
     - `sounds/` 文件夹
     - `.github/workflows/build-apk.yml`

4. **触发构建**
   - 在仓库页面点击 "Actions"
   - 选择 "Build Android APK"
   - 点击 "Run workflow" → 绿色 "Run workflow" 按钮

5. **下载 APK**
   - 等待约 30-60 分钟
   - 完成后在 Actions 页面下载 Artifacts

## 🔧 方案3：修复 WSL 网络问题（如果一定要用 WSL）

### 尝试1：修改 hosts 文件
1. 以管理员身份打开记事本
2. 打开文件：`C:\Windows\System32\drivers\etc\hosts`
3. 在末尾添加：
   ```
   185.199.108.133 raw.githubusercontent.com
   185.199.109.133 raw.githubusercontent.com
   185.199.110.133 raw.githubusercontent.com
   185.199.111.133 raw.githubusercontent.com
   ```
4. 保存
5. 在 PowerShell 运行：`ipconfig /flushdns`
6. 再试：`wsl --install`

### 尝试2：离线安装 WSL
1. 下载 WSL 更新：https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi
2. 下载 Ubuntu 22.04：https://aka.ms/wslubuntu2204
3. 先安装 WSL 更新包，再安装 Ubuntu
